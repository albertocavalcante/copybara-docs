Globs define file patterns for filtering.

## Syntax

```starlark
glob(include, exclude)
```

## Pattern Characters

| Pattern  | Matches                      |
| -------- | ---------------------------- |
| `*`      | Any characters except `/`    |
| `**`     | Any characters including `/` |
| `?`      | Single character             |
| `[abc]`  | Character class              |
| `[!abc]` | Negated character class      |

## Examples

### Include Only

```starlark
glob(["src/**", "docs/**"])
```

### Include and Exclude

```starlark
glob(
    include = ["**"],
    exclude = ["**/internal/**"],
)
```

### By Extension

```starlark
glob(["**/*.py", "**/*.pyi"])
```

### By Directory

```starlark
glob(["src/**"])
glob(["src/*"])  # One level only
```

### Exclude Multiple

```starlark
glob(
    include = ["**"],
    exclude = [
        "**/test/**",
        "**/*_test.go",
        "**/testdata/**",
    ],
)
```

## Usage Contexts

### origin_files

```starlark
origin_files = glob(["src/**"])
```

### destination_files

```starlark
destination_files = glob(
    include = ["**"],
    exclude = ["README.md"],
)
```

### Transformation paths

```starlark
core.replace(
    before = "old",
    after = "new",
    paths = glob(["**/*.md"]),
)
```

### core.remove

```starlark
core.remove(glob(["**/*.bak"]))
```

## Advanced Patterns

### Match Root Files Only

```starlark
glob(["*.md"])           # Only root-level .md files
glob(["**/*.md"])        # All .md files recursively
```

### Match Specific Depth

```starlark
glob(["src/*/*.py"])     # Python files exactly 1 level under src/
glob(["src/**/*.py"])    # Python files at any depth under src/
```

### Multiple Extensions

```starlark
glob(["**/*.{js,ts,jsx,tsx}"])  # NOT supported - use list instead
glob(["**/*.js", "**/*.ts", "**/*.jsx", "**/*.tsx"])  # Correct
```

### Hidden Files

```starlark
glob(["**/.*"])          # All hidden files (starting with .)
glob(["**/.github/**"])  # .github directory contents
```

## Common Pitfalls

### Pitfall: Missing `**` for Recursive Match

```starlark
# WRONG: Only matches src/file.py, not src/sub/file.py
glob(["src/*.py"])

# CORRECT: Matches all .py files under src/
glob(["src/**/*.py"])
```

### Pitfall: Exclude Order Matters

Excludes are processed after includes. An excluded path cannot be re-included.

```starlark
glob(
    include = ["**"],
    exclude = ["internal/**"],
)
# internal/important.txt is EXCLUDED even if you wanted it
```

### Pitfall: Trailing Slashes

Glob patterns should NOT have trailing slashes:

```starlark
# WRONG
glob(["src/"])

# CORRECT
glob(["src/**"])
```

## Performance Tips

- Be as specific as possible with include patterns
- Exclude large directories you don't need (e.g., `node_modules`, `.git`)
- Prefer `["src/**"]` over `["**"]` + exclude when possible
